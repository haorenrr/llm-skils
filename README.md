## 简介

这个库提供了生产级别的 AI 技能，通过自动化工作流增强模型能力。这些技能兼容多个 LLM 平台，包括 Claude、Google Gemini 等。

### 什么是技能？

技能是一个专业工具，整合了：
- **角色定位**：特定的人设和专业领域
- **核心准则**：质量和一致性的指导方针
- **自动化脚本**：基于 Python 的执行工具
- **工作流程**：循序渐进的任务处理流程

## 可用技能

### 🎬 风景猎人 (Short Video Hunter)
**版本**: 1.2.0 | **状态**: ✅ 已激活

专业寻找和下载治愈系短视频，优化情感疗愈和视觉灵感。

#### 主要特性
- **治愈系搜索**：专注于风景名胜、乡村风光和公路驾驶题材
- **智能全屏模式 (Smart Crop)**：针对横屏视频，通过 AI 视觉自动识别重心并执行 9:16 裁剪
- **重心追踪**：利用 LLM 视觉分析定位主体（太阳、山峰等），支持动态镜头平滑跟随
- **画质红线**：原生点对点像素提取，严禁拉伸，确保 1080p/4K 的极致清晰度
- **智能稳定性过滤**：自动识别并丢弃晃动剧烈的镜头，确保画面稳如云台
- **批量处理**：高效下载和处理多个视频

#### 智能全屏模式说明
当你在需求中提及 **“竖屏”**、**“9:16”**、**“全屏”** 或 **“满屏”** 时，技能将自动从“背景模糊模式”切换至“智能全屏模式”：
1. **静默侦察**：AI 会预先分析视频帧，锁定视觉能量中心。
2. **精准对齐**：自动计算 X 轴偏移，将核心主体（如日出的太阳）精准置于竖屏中央。
3. **动态跟随**：如果视频有左右摇镜，裁剪框将平滑跟随主体移动。
4. **无损输出**：不进行任何图像拉伸，保持最高像素锐度。

#### 支持主题
- 🏔️ **大美风景**：壮阔山脉、无尽海平线、辽阔视野
- 🚗 **开放之路**：电影级公路驾驶，奔向山脉、荒漠或海边
- 🌾 **田园宁静**：祥和乡村、晨雾、宁谧村庄
- ☁️ **天空云艺**：治愈的云卷云舒、星空美景

#### 应用场景
- 为冥想应用制作治愈系视频集
- 为健康类内容频道寻找素材
- 为旅游创作者采集景观视频
- 为放松和正念应用精选视觉内容
- 构建自然纪录片库

## 快速开始

### 前置条件

AI（Claude、Gemini 等）会处理所有技术细节，你只需要：

- 支持此技能的 LLM（Claude、Gemini 等）
- 网络连接用于内容搜索
- 足够的存储空间

### 三步上手

1. **打开 Claude cli 或 Gemini cli**（或其他支持的 LLM）

2. **启用风景猎人技能**（如果平台需要）

3. **用自然语言描述需求**：
   ```
   我想要一些治愈系驾驶视频 - 
   在空旷的公路上开向山脉，很平静的感觉，
   每个视频大概15秒。能给我5-8个吗？
   ```

4. **AI 自动处理**→ 搜索、下载、处理、优化、交付

### 通过 AI 对话使用风景猎人

只需用自然语言描述你的需求，AI 会完成全部工作：

#### 场景1：个人压力缓解库
**你对 Claude/Gemini 说：**
> "我想收集一些睡前看的治愈视频。特别想要那种在空旷高速公路上开车，远处是山或海，很宁静的感觉。能给我5个高质量竖屏视频，每个15秒左右吗？"

**AI 的处理过程：**
- 搜索治愈系驾驶素材
- 筛选原生 9:16 竖屏格式
- 验证质量和时长
- 下载并优化文件
- 按主题和心情分类
- 返回现成可用的视频和描述

**输出结果**：5 个竖屏 MP4 文件 + 情绪标签

---

#### 场景2：冥想应用内容创作者
**你对 Gemini 说：**
> "我在做一个冥想应用的内容。需要20个短视频，每个不超过20秒，要包括：
> - 辽阔的风景（山脉、地平线、山谷）
> - 乡村清晨场景
> - 沿海风光，人烟稀少
> 全部必须是1080p竖屏格式。能帮我整理好吗？"

**AI 的处理过程：**
- 多主题变体搜索
- 1080p 最低分辨率筛选
- 时长和宽高比验证
- 批量视频处理
- 按主题分文件夹整理
- 为每个视频生成元数据（心情、主题、治愈功效）

**输出结果**：20 个视频分别存放在 `风景/`、`乡村/`、`海洋/` 文件夹 + 元数据表

---

#### 场景3：旅游频道视频策展
**你对 Claude 说：**
> "我在做一个叫《通往宁静之路》的 YouTube 合集。帮我找一些不同地区的景观驾驶视频——最好是在空旷的公路上开向山脉、荒漠或海滩的感觉。要能表现自由和辽阔的那种。每个8-12秒，我需要10个视频。"

**AI 的处理过程：**
- 搜索"电影级驾驶"相关内容
- 优先选择开放之路和自然风景
- 按情绪冲击力筛选
- 验证技术指标
- 生成编辑就绪格式
- 建议最优情绪流程排序

**输出结果**：10 个视频按最优情绪序列排列 + 时间提示

---

### 工作原理

当你提出请求时，AI 技能会：

1. **理解意图** → 解析你的自然语言需求
2. **生成关键词** → 将主题映射到优化的搜索查询
3. **搜索和筛选** → 查询符合条件的内容
4. **验证质量** → 检查分辨率、时长、格式、宽高比
5. **处理媒体** → 提取、剪辑、优化视频
6. **整理输出** → 组织文件和生成元数据
7. **交付结果** → 返回即用型视频集合

**技术栈**（全由 AI 自动处理）：
- 搜索：`yt-dlp --js-runtimes node`
- 处理：`video_processor.py` + ffmpeg
- 验证：时长、分辨率、宽高比检查
- 输出：优化的竖屏 MP4 + 元数据

## 本地部署（开发者和自建）

如果你要在本地部署或集成这个技能：

1. **克隆仓库**
   ```bash
   git clone https://github.com/haorenrr/llm-skils.git
   cd llm-skills/short-video-hunter
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **验证安装**
   ```bash
   python scripts/video_processor.py --help
   ```

**技术栈**：
- Python 3.8+
- `yt-dlp`（视频搜索）
- `ffmpeg`（视频处理）
- `video_processor.py`（协调脚本）

## 自定义和微调

### 常见需求表达方式

| 你的需求 | 怎么说 | AI 会处理 |
|---|---|---|
| 最平静的治愈视频 | "我要最宁静、最治愈的视频" | 优先选择平和主题，过滤刺激内容 |
| 快节奏内容 | "给我充满活力、鼓舞人心的驾驶视频" | 选择动态场景、快速剪辑 |
| 特定季节或心情 | "冬天的雪山场景" | 语义搜索季节主题 |
| 集合内容 | "我需要20个视频的合集" | 批量处理，按主题分类 |
| 特定时长 | "每个视频10-12秒" | 精确剪裁到指定时长 |
| 手机优化 | "我需要竖屏视频发抖音" | 优先 9:16 原生格式 |
| 高清品质 | "只要1080p或更高" | 处理前验证分辨率 |
